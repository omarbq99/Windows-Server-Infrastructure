Projet 01 : Basic Active Directory Lab

Déploiement d’une infrastructure Windows Server en environnement virtuel.

⸻

Présentation

Ce projet présente la mise en place d’une infrastructure Windows Server basée sur Active Directory.

L’objectif principal est de simuler un environnement d’entreprise comprenant la gestion des utilisateurs, des services réseau et des stratégies de sécurité.

⸻

Objectifs

* Déployer un contrôleur de domaine Active Directory
* Configurer les services DNS et DHCP
* Créer une structure d’unités d’organisation (OU)
* Gérer les utilisateurs et les groupes
* Mettre en place des partages de fichiers sécurisés
* Appliquer des stratégies de groupe (GPO)
* Valider le fonctionnement de l’infrastructure

⸻

Technologies utilisées

* Windows Server 2019
* Active Directory Domain Services (AD DS)
* DNS Server
* DHCP Server
* Group Policy (GPO)
* NTFS Permissions
* PowerShell
* Oracle VirtualBox

⸻

Architecture (résumé)

* Domaine : bqlab.local
* Serveur : DC1 (192.168.10.10)
* Clients : machines Windows en DHCP
* Réseau : 192.168.10.0/24
